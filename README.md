##Viz

An easy tool to map out your development lifecycle!
For documentation, to demo to stakeholders, for audits and more.

Features:

- Home screen
  - Shows flows, types and quick buttons
- Create/edit flows
  - Flow name
  - Description
  - Tech e.g. react/py/php etc.
  - Stages
    - Isolated or parallel steps with types (created by user)
- Create/edit types
  - Name/colour
- Visual flow (view your flow, in a connecting node lifecycle)
  - Connecting nodes as created by user
  - Node type key
  - Coloured/named nodes
  - Isolatted or parallel node flows
  - Export
    - To JSON
    - To image (png)
